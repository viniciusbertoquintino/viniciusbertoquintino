# AI Assistant Platform — LLMs, RAG & Agents

Plataforma corporativa de IA Generativa para análise de documentos, automação de atendimento e suporte operacional usando LLMs, RAG e agentes inteligentes.

## Principais recursos

- Chat com documentos usando RAG
- Upload e indexação de PDFs, Word e Excel
- Busca semântica com embeddings
- Agentes inteligentes para análise de documentos e chamados
- Integração simulada com Jira/Movidesk
- Observabilidade básica de prompts, tokens e respostas
- Docker Compose para execução local

## Stack

Python, FastAPI, PostgreSQL, pgvector, LangChain, LangGraph, OpenAI/Azure OpenAI, React, Next.js, Docker.
